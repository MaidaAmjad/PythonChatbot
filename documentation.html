Project Overview

Nocturnal Intelligence is a high-performance, premium tier AI chatbot built on the Gemini architecture. Crafted for speed, aesthetic excellence, and utility, it integrates cutting-edge image generation and deep-thought conversation capabilities within a glassmorphism workspace.

Key Features

Neural Chat

Powered by gemini-1.5-flash, offering zero-latency responses and deep contextual understanding.

Neural Artist

Integrated drawing engine capable of creating stunning visuals via natural language prompts like "Draw a neon city".

Workspace Context

Automatic session naming and persistent chat history stored locally for total privacy.

Tech Stack

  • Backend: Python 3.x with Streamlit API Interface
  • Engine: Google Generative AI (Gemini Flash Architecture)
  • Visuals: Pollinations.ai Diffusion Network
  • Persistence: JSON-based Local History Storage
  • UI Engine: Custom Glassmorphism CSS Framework

Installation Guide

1. Clone & Setup

git clone https://github.com/MaidaAmjad/PythonChatbot.git
pip install -r requirements.txt

2. Configure Environment

Create a .env file in the root directory:

GEMINI_API_KEY=your_google_ai_studio_api_key

3. Launch Terminal

streamlit run app.py
